# HTML3D
HTML3D is a JavaScript library for creating 3D scenes using CSS 3D transforms.

See the demo at: https://html3d.com

![HTML3D demo](./html3d_demo.png?raw=true "HTML3D demo")

**src/index.html**<br>
index.html is the homepage of html3d.com. It showcases a recursive game-within-a-game.
<br />
All the code is in the html file. You can use this file as a starting point for creating your own 3D scenes.

**src/HTML3D.js**<br>
Use HTML3D.js in your projects.
